# Akan Name Generator

**Author:** Eric Gitika


## Description

The Akan Name Generator is a web application that assigns a traditional Ghanaian Akan name to a user based on their date of birth. In Akan culture, every child is given a name according to the day of the week they were born. The app takes a user's birthdate and gender as input, calculates the day of the week using a mathematical formula, and displays the corresponding Akan name along with its cultural meaning.


### Running Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/EricGitika/Akan-Name-Generator.git
   ```

2. **Navigate into the project folder**
   ```bash
   cd Akan-Name-Generator
   ```
